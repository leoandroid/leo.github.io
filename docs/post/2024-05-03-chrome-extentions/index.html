1. Ontab 标签整理

Save up to 95% memory and reduce tab clutter Whenever you find yourself with too many tabs, click the OneTab icon to convert all of your tabs into a list. When you need to access the tabs again, you can either restore them individually or all at once.

iTab新标签页,强大的浏览器新标签页,可以自定义自己网站捷径, 以及搜索引擎. 并且内置多个app等功能. 可以设置捷径图标样式,一个让你不受广告干扰的个性化卡片式起始页插件，创建和编辑属于您自己的浏览器标签页，精美日历、炫酷天气、每日头条、海量壁纸、常用网址随心订制

Dark mode for every website. Take care of your eyes, use dark theme for night and daily browsing.

Save a complete page into a single HTML file SingleFile is an extension that helps you to save a complete page (with CSS, images, fonts, frames, etc.) as a single HTML file.

Adds code intelligence to GitHub, GitLab, and other hosts: hovers, definitions, references. For 20+ languages.
